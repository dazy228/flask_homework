# flask_homework
Описание
Проект содержит несколько API-методов, созданных с помощью Flask. Эти методы позволяют выполнить некоторые задачи, используя сторонние библиотеки.

Требования
Для запуска проекта необходим Python версии 3.x и следующие библиотеки:

Flask
requests
pandas
faker
Для установки всех необходимых библиотек можно использовать команду:
pip install -r requirements.txt

Запуск
Чтобы запустить проект, необходимо запустить файл app.py следующей командой:
python app.py
или
FLASK_APP=app FLASK_DEBUG=1 TEMPLATES_AUTO_RELOAD=True flask run

API-методы
GET /
Возвращает сообщение "Hello, World!".

GET /requirements/
Возвращает список требуемых библиотек.

GET /generate-users/
Генерирует список пользователей с помощью библиотеки faker. Возвращает 100 случайно сгенерированных имён и адресов электронной почты.

GET /generate-users/<int:count>/
Генерирует список пользователей с помощью библиотеки faker. Возвращает <count> случайно сгенерированных имён и адресов электронной почты.

GET /mean/
Вычисляет средний рост и средний вес людей на основе данных в файле hw.csv. Возвращает значения среднего роста и среднего веса в сантиметрах и килограммах соответственно.

GET /space/
Получает данные о количестве космонавтов, находящихся в настоящий момент в космосе, с помощью API-источника http://api.open-notify.org/astros.json. Возвращает количество космонавтов в настоящий момент.
